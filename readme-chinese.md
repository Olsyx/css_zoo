# CSS動物園 :pig: :frog: :bee: :beetle: :lion:
用單純的HTML及CSS來組合出不同動物的圖案。
# 指南
+ 以該動物之英文命名HTML檔
+ 用單純的HTML跟CSS做出動物的圖像
+ 不能使用圖片
+ 加上作者的資料
+ 跟著[hacktoberfest code of conduct] (https://docs.google.com/document/d/1gFKOhyUqMZzrZcbq8A_TpO5x9J9HK6agv70awCH8pyI/edit)的指示
